# Harvia Fenix (Home Assistant)

Custom Integration for Harvia Fenix / Xenio WiFi (cloud).

## Installation (HACS)
1. HACS → Integrations → “Custom repositories”
2. Add this repository URL as type **Integration**
3. Install “Harvia Fenix”
4. Restart Home Assistant
5. Settings → Devices & Services → Add Integration → Harvia Fenix

## Configuration
Login using your Harvia account (email/password).

## Notes
- Uses Harvia cloud APIs
- iot_class: cloud_polling
