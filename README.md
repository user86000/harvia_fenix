Harvia Fenix
Home Assistant custom integration for Harvia Fenix sauna controllers.
Features
Adds Harvia Fenix sauna control to Home Assistant
Local polling integration
Config flow based setup
Suitable for installation through HACS as a custom repository
Installation with HACS
Open Home Assistant.
Go to HACS.
Open the three-dot menu.
Choose Custom repositories.
Add this repository URL:
`https://github.com/user86000/harvia_fenix`
Select category Integration.
Install Harvia Fenix.
Restart Home Assistant.
Manual installation
Copy the folder:
```text
custom_components/harvia_fenix
```
to your Home Assistant configuration directory:
```text
/config/custom_components/harvia_fenix
```
Then restart Home Assistant.
Configuration
After installation:
Go to Settings → Devices & services.
Click Add integration.
Search for Harvia Fenix.
Follow the setup dialog.
